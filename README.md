# To-Do List App

This is a simple and elegant to-do list application built with HTML, CSS, and JavaScript. It helps you keep track of your tasks, and your data is saved in your browser's local storage.

## Features

*   **Add Tasks:** Quickly add new tasks to your list.
*   **Delete Tasks:** Remove tasks you no longer need.
*   **Edit Tasks:** Click on the "Edit" button to modify a task.
*   **Mark as Completed:** Click on a task to toggle its completed status.
*   **Filter Tasks:** View all, active, or completed tasks.
*   **Local Storage:** Your tasks are saved in your browser, so you won't lose them when you close the page.

## How to Use

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/neruban007/todo-list-app.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd todo-list-app
    ```
3.  **Open `index.html` in your browser:**
    You can do this by double-clicking the file or by running a local server.

## Code Structure

*   **`index.html`:** The main HTML file that defines the structure of the application.
*   **`style.css`:** The CSS file that styles the application, giving it a modern, dark theme.
*   **`script.js`:** The JavaScript file that contains all the logic for the application, including adding, deleting, editing, and filtering tasks.
